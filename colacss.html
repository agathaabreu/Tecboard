CSS

A regra básica é informar ao navegador qual elemento desejamos modificar e, em seguida, o que exatamente queremos alterar nesse elemento.
Por exemplo h1: adiciona atributo style e modificamos o tamanho da fonte do título escrevendo font-size , e o VSCode nos oferece um autocomplete. Podemos usar valores como large, medium, small, mas é mais prático trabalhar com valores em pixels. Exemplo: font-size como 40px.

Além do tamanho da fonte, podemos modificar a cor. Em vez de font-color, usamos apenas color.

Podemos também alterar a família de fontes com font-family, escolhendo, por exemplo, Arial.

Se desejarmos espaçar mais as letras, utilizamos a propriedade letter-spacing, definindo um valor em pixels, como exemplo 5px.

Podemos também ajustar o espaçamento entre linhas com line-height. Se a letra é 25 pixels por exemplo, um espaçamento de 10 pixels pode ser insuficiente, então ajustamos para 35 ou 50 pixels, conforme necessário.

Existem três formas de aplicar CSS: inline (na mesma linha do código), interno e externo. O CSS inline é descontinuado devido à poluição de código. O CSS interno, embora melhor, ainda deixa o arquivo longo. A prática recomendada é o CSS externo.

Criando css

Todo navegador aplica um conjunto de estilos próprios aos elementos HTML, e precisamos informar ao navegador que não queremos esses estilos, para que possamos trabalhar com uma base limpa. Existem várias maneiras de fazer isso, o que chamamos de reset CSS.
Para começar o reset CSS, utilizamos o seletor universal *, que aplica estilos a todos os elementos.
*{ Depois removemos as margens padrão que o navegador aplica: margin:0 e, em seguida, removemos também o padding.

O próximo passo é definir um estilo para o fundo.Para aplicar cores Podemos usar uma cor nomeada, como h1 { color: red; }, ou um formato hexadecimal, composto por seis dígitos, onde cada par representa a quantidade de vermelho, verde e azul. O hexadecimal do vermelho é #FF0000. Podemos também usar o formato RGB, como rgb(255, 0, 0). A forma mais usada é o hexadecimal. o hexadecimal do fundo é copiado e aplicado ao body no CSS, usando a propriedade background-color.

Por exemplo, criamos uma classe chamada "cor-branca.{ e colocamos color: #FFFFFF.fechachave- Criamos uma classe e precisamos ir ao elemento, criar um atributo class e colocar o valor como cor-branca, que é a classe criada. Reset css

Informar ao navegador que não queremos que padrões de estilos automáticos sejam aplicados, e assim começamos a estilizar a página de uma base limpa.

Existem diferentes abordagens para fazer isso:
Reset tradicional: zera praticamente todos os estilos (como margens, paddings e fontes), deixando tudo uniforme.
Normalize.css: em vez de remover totalmente os estilos, harmoniza as diferenças entre navegadores, mantendo padrões úteis.
CSS moderno: algumas pessoas preferem criar seus próprios resets minimalistas, ajustando apenas o que realmente impacta o projeto.
